# mapa-electoral-chiapas
Visor geográfico interactivo de casillas electorales del estado de Chiapas permite visualizar y consultar información territorial y resultados electorales mediante mapas web dinámicos.  Desarrollado por Baldemar Maza León para análisis y visualización geoespacial de datos electorales. © 2026 Baldemar Maza León. Todos los derechos reservados.
